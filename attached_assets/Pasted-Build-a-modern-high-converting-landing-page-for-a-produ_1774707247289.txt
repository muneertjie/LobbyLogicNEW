Build a modern, high-converting landing page for a product called "LobbyLogic".

Goal:
Convert service-based businesses (dentists, doctors, salons, spas, clinics, etc.) into paying customers by clearly showing how LobbyLogic increases revenue from existing foot traffic.

Tone & Style:

* Premium, clean, modern (Apple / Stripe-inspired design)
* Dark + light contrast sections
* Smooth scrolling
* Bold headlines, minimal text
* Focus on revenue, psychology, and ROI
* Subtle animations (fade-ins, hover effects)

Tech:

* Use HTML, CSS, and JavaScript (or React if preferred)
* Mobile responsive
* Fast loading
* Include smooth scrolling navigation

Structure the page as follows:

Design Requirements:

* Use large bold fonts for headlines
* Plenty of white space
* Smooth transitions between sections
* Subtle animations on scroll
* Buttons should stand out clearly
* Include icons or simple illustrations where helpful

Conversion Focus:

* Every section should reinforce revenue growth
* Avoid technical jargon
* Keep messaging simple and powerful
* Make the user feel like they are losing money by not using this

Optional Enhancements:

* Add a sticky header with CTA button
* Add a floating “Book Demo” button
* Add a simple calculator (optional) where users input daily customers and see potential revenue increase
